[web2 CSS - 13. 미디어 쿼리 써먹기]

스크린의 사이즈가 작아질 때 페이지의 모습이 이상한 것을 수정하는 방법
@media(max-width:800px){
	#grid{
		display: block;
	}
}

##
화면이 800px보다 작을 때 block으로 처리했기 때문에 자연스럽게 글씨가 밑으로 내려온다. 

border right라는 속성 때문에 '선'이 존재. -> 이 선을 삭제시켜주려면, 
ol{
	border-right:1px solid gray;
를 복사해서 가져온 후 border-right:none; 으로 바꿔준다. 

h1태그의 테두리도 같은 방법으로 삭제해 준다. 
##


[web2 CSS - 14. CSS 코드의 재사용]

페이지 크기가 800px이상일 때는 선 안으로 텍스트가 들어와야하는데 왜 안들어오는지 모르겠다ㅜㅜ 코드는 똑같은데 뭐가 문제일까... 아직 해결을 못했다. 

모든 페이지에 같은 코드 -> 중복된 코드 제거가 중요!!
style 태그 안에 들어있는 css코드를 복사하여 style.css파일을 만들어 붙여넣기 해준다. 각 파일에 중복되어 있는 style 태그 코드들은 삭제! 
코드를 삭제한 곳에 link를 통하여 style 태그와 연결시켜준다! 

효율적으로 페이지 완성!!

장점
1. link를 통해 파일을 연결하면 각각의 페이지 style 코드가 같은 것을 바로 확인할 수 있지만 css코드를 직접 복사붙여넣기 하면 코드가 같은 것을 확인하기 어렵다!
2. 웹페이지를 다운받을 때도 경제적이다!
